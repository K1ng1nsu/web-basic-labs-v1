제목 태그

HTML은 여섯 단계의 제목 태그를 제공합니다.

h1 태그: 가장 큰 제목

h2 태그: 두 번째 제목

h3 태그: 세 번째 제목

h4 태그: 네 번째 제목

h5 태그: 다섯 번째 제목
h6 태그: 가장 작은 제목

제목 태그는 문서의 구조를 나타내는 데 중요하며, 검색 엔진이 웹 페이지의 콘텐츠를 이해하는 데 도움을 줍니다.

단락 및 줄바꿈

p 태그는 단락을 나타냅니다. 단락은 하나의 주제나 생각을 담고 있는 텍스트 블록입니다.

이것은 두 번째 단락입니다. HTML에서는 단순히 Enter 키로 줄을 바꿔도 브라우저는 한 줄로 표시합니다.
하지만 br 태그를 사용하면 강제로 줄바꿈을 할 수 있습니다.

hr 태그는 수평선을 만들어 콘텐츠를 구분합니다.

텍스트 서식

strong 태그는 중요한 텍스트를 강조합니다. 일반적으로 굵게 표시됩니다.

em 태그는 강조된 텍스트를 나타냅니다. 일반적으로 기울임꼴로 표시됩니다.

mark 태그는 하이라이트된 텍스트를 나타냅니다.

small 태그는 작은 텍스트를 나타냅니다.

del 태그는 삭제된 텍스트를 나타냅니다.

ins 태그는 추가된 텍스트를 나타냅니다.

sub 태그와 sup 태그는 각각 아래 첨자와 위 첨자를 나타냅니다.

code 태그는 코드 조각을 나타냅니다.

pre 태그는
    공백과
        줄바꿈을
            그대로 유지합니다.

인용구

인라인 인용구는 q 태그를 사용합니다.

blockquote 태그는 블록 레벨 인용구를 나타냅니다. 일반적으로 들여쓰기가 적용되며, 여러 단락이나 긴 인용구에 사용됩니다.

cite 태그는 작품 제목과 같은 인용 출처를 나타냅니다.

주요 텍스트 태그 요약

태그 설명
<h1> - <h6> 제목 태그
<p> 단락 태그
<br> 줄바꿈 태그
<hr> 수평선 태그
<strong> 중요한 텍스트 강조
<em> 강조된 텍스트
<mark> 하이라이트된 텍스트
<blockquote> 블록 레벨 인용구
<q> 인라인 인용구
<pre> 서식이 있는 텍스트
