    자바스크립트에서 this의 값이 달라지는 경우는?

    정답 1번: 함수 호출 방식에 따라
  • Q2. ×
    setTimeout(() => { ... }, 1000)은 무엇을 의미하는가?
    내 선택 3번: 즉시 실행한다
    정답 2번: 1초 후에 한 번 실행한다
  • Q3. ×
    자바스크립트에서 함수를 값처럼 전달할 수 있는 이유는?
    내 선택 3번: 함수는 배열이기 때문
    정답 1번: 함수도 객체이기 때문
  • Q4. ×
    fetch 함수로 데이터를 받아올 때 주로 사용하는 문법은?
    내 선택 3번: setInterval
    정답 2번: Promise
  • Q5. ×
    아래 중 'false'로 평가되지 않는 값은?
    내 선택 3번: undefined
    정답 4번: 'false'
  • Q6. ×
    배열의 길이를 구하는 속성은?
    내 선택 4번: index
    정답 3번: length
  • Q7. ×
    typeof null의 결과는 무엇인가?
    내 선택 4번: 'number'
    정답 1번: 'object'
  • Q8. ×
    다음 중 원시 타입(Primitive type)이 아닌 것은?
    내 선택 4번: boolean
    정답 3번: object
  • Q9. ×
    forEach 메서드의 올바른 사용법은?
    내 선택 4번: arr.forEach = function() { ... }
    정답 1번: arr.forEach(item => { ... })
  • Q10. ○
    자바스크립트에서 변수를 선언할 수 없는 키워드는?
